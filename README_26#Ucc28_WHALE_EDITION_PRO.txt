변경 내용:
- 황금 고래 브랜드를 HOME 메인/사이드바/출석 보상에 통일 적용

- 오늘 매매 체크보드 강조: 알트 신호 / BTC.D / OI 급증 / 핫코인 / 리스크
- 출석 보상 페이지를 프리미엄 리워드 센터 느낌으로 강화
- 기존 app.py, DB, 시장 분석, 게임, 랭킹, 지갑, 채굴장 기능 유지

실행:
1) 압축 풀기
2) run_26_WHALE_EDITION_PRO.bat 실행 또는 cmd에서 python app.py
3) http://127.0.0.1:5000 접속
