CBK 23차 HOME PRO ULTIMATE 전체복사본

실행 방법
1. 압축을 풀기
2. run_23_HOME_PRO.bat 실행
   또는 cmd에서 python app.py 실행
3. 브라우저에서 http://127.0.0.1:5000 접속

포함 파일 확인
- app.py
- index.html
- cryptobreaker_cbp.db
- requirements.txt
- .env.example
- assets/backgrounds 배경 7종
- sounds 효과음

적용 내용
- 홈 화면 고급형 리디자인
- 한국어 메뉴 적용
- 검정/보라/금색 네온 글래스 UI
- 시장 현황, 실시간 인기 코인, 빠른 이동, 커뮤니티 현황 카드
- 기존 대시보드/게임/출석/랭킹/레퍼럴/지갑/채굴장 기능 유지
